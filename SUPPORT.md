# Support

This is a personal **teaching / demonstration** project, maintained on a
best-effort basis. Here's how to get help or engage:

- **Questions / ideas / show-and-tell** → open a
  [Discussion](https://github.com/jkindrix/overengineered-todo/discussions).
- **Bugs / feature requests** → open an
  [Issue](https://github.com/jkindrix/overengineered-todo/issues/new/choose)
  using one of the templates.
- **Security issues** → see [SECURITY.md](SECURITY.md) (report privately).
- **Understanding the codebase** → start with the
  [docs site](https://jkindrix.github.io/overengineered-todo/), especially
  *"50 lines vs. this"* and the *code tour*.

There is no SLA. Response times depend on availability.
